# Sound Files

This folder contains sound effects for the game.

## Required Files

- `correct.mp3` — Correct answer sound
- `wrong.mp3` — Wrong answer sound
- `win.mp3` — Win game sound
- `tick.mp3` — Timer tick sound

## Placeholder

For now, create empty files or use free sound effects from:
- https://freesound.org/
- https://www.zapsplat.com/
- https://mixkit.co/free-sound-effects/

## Adding Sounds

1. Download free sound effects
2. Place them in this folder
3. Rename them to match the filenames above
4. Sounds will be loaded automatically by SoundManager
